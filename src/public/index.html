Node.js School assignment API info

Base URL

Root Path:

https://educase-assignment-q4ex.onrender.com/

This API is built using Node.js, Express.js, and MySQL to manage school data and sort schools based on proximity.

Database Structure (MySQL)

Table: schools

id (Primary Key)
name (VARCHAR)
address (VARCHAR)
latitude (FLOAT)
longitude (FLOAT)

Add School API

Endpoint: POST https://educase-assignment-q4ex.onrender.com/api/addschool

Functionality: Validates input and inserts a new school into the database.

Payload:

{
  "name": "ABC School",
  "address": "Bangalore",
  "latitude": 12.9716,
  "longitude": 77.5946
}

Validation: All fields must be non-empty and of correct data type.

List Schools API

Endpoint: GET /listSchools

Parameters: latitude, longitude

Example: https://educase-assignment-q4ex.onrender.com/api/listSchools?latitude=12.9716&longitude=77.5946

Functionality: Fetches all schools and returns them sorted by distance from the user’s location.

Sorting Logic: Uses geographical distance calculation to sort nearest schools first.

Deployment & Testing

  • Deployed on render and used railone for MySQL to make it live.
  • Created a Postman collection for testing both endpoints

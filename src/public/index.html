School Management API

This API allows you to add schools and fetch them sorted by proximity.

Add School

Endpoint: POST /api/addSchool

Body:

{
  "name": "ABC School",
  "address": "Bangalore",
  "latitude": 12.9716,
  "longitude": 77.5946
}

List Schools (Nearest First)

Endpoint: GET /api/listSchools?latitude=12.97&longitude=77.59
